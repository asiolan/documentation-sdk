# documentation-sdk
My documentation-sdk project

## Purpose
This project serves as a comprehensive SDK for managing documentation. 

## Features
- Create, read, update, and delete documentation files.
- Support for multiple formats (Markdown, HTML).

## Installation
To install the SDK, run:
```bash
pip install documentation-sdk
```

## Usage
```python
from documentation_sdk import Documentation

doc = Documentation()
doc.create('example.md', content='This is an example.')
```
